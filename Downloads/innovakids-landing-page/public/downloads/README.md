# Instrucciones para subir el Kit del Padre Moderno

## Pasos para activar la descarga del PDF:

1. **Sube tu PDF aquí** en esta carpeta `public/downloads/`
2. **Nombre del archivo:** Debe llamarse exactamente `kit-padre-moderno.pdf`
3. **Ruta final:** `public/downloads/kit-padre-moderno.pdf`

## ¿Cómo funciona?

Cuando un padre complete el formulario del Lead Magnet:
- ✅ Su email se guarda automáticamente en Supabase
- ✅ El PDF se descarga inmediatamente desde esta ubicación
- ✅ Puedes ver todos los leads capturados en tu base de datos Supabase
